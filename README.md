### Prerequisites

```
brew install stow
```

### Installation

```
chmod +x ./install
./install
```

### First iTerm & tmux setup

```
brew install fzf
brew install tmux
```

Go to iTerm -> Settings -> Profiles -> Send text at start:

```
tmux attach -t default || tmux new -s  default
```

### Install Espanso

```
brew install espanso
```
